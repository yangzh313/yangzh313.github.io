Linux端口转发的几种常用方法

ncat#

安装ncat

yum install nmap-ncat

监听本机11180端口，将数据转发到179.19.3.111得8080端口

ncat --sh-exec "ncat 179.19.3.111 8080" -l 11180  --keep-open
